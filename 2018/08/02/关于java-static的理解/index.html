Static keyword

一开始做这些题目的时候一脸懵逼：但其实只要遵循下面几条规则就能轻易的做出来

  • 被static修饰的所有东西都是在创建对象之前，随着类的加载并且只会执行一次
  • 非static修饰的每次创建对象都会执行
  • 构造方法执行之前一定会先初始化普通变量与构造代码块
  • static代码块与static成员变量是一个级别，谁在前先执行谁构造代码块与成员变量一个级别
